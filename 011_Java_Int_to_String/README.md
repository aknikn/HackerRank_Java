# 011: [Java Int to String](https://www.hackerrank.com/challenges/java-int-to-string/problem "https://www.hackerrank.com/challenges/java-int-to-string/problem")
- level: Easy
- language: Java


## Description of the problem
You are given an integer n, you have to convert it into a string.<br>
Please complete the partially completed code in the editor.<br>
If your code successfully converts n into a string s the code will print "Good job".<br>
Otherwise it will print "Wrong answer".<br>
n can range between -100 to 100 inclusive.

### Sample Input
> 100

### Sample Output
> Good job
