# 027: [Java BigInteger](https://www.hackerrank.com/challenges/java-biginteger/problem "https://www.hackerrank.com/challenges/java-biginteger/problem")
- level: Easy
- language: Java


## Description of the problem
In this problem, you have to add and multiply huge numbers!<br>
These numbers are so big that you can't contain them in any ordinary data types like a long integer.<br><br>

Use the power of Java's BigInteger class and solve this problem.

### Imput Format
There will be two lines containing two numbers, *a* and *b*.

### Constraints
*a* and *b* are non-negative integers and can have maximum 200 digits.

### Output Format
Output two lines.<br>
The first line should contain *a*+*b*, and the second line should contain *a* \* *b*.<br>
Don't print any leading zeros.

### Sample Input
> 1234<br>
> 20

### Sample Output
> 1254<br>
> 24680

### Explanation
1234 + 20 =  1254<br>
1234 * 20 = 24680
