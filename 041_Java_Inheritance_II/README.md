# 041: [Java Inheritance II](https://www.hackerrank.com/challenges/java-inheritance-2/problem "https://www.hackerrank.com/challenges/java-inheritance-2/problem")
- level: Easy
- language: Java

## Description of the problem
Write the following code in your editor below:

1. A class named Arithmetic with a method named add that takes 2 integers as parameters and returns an integer denoting their sum.<br>
2. A class named Adder that inherits from a superclass named Arithmetic.<br>
Your classes should not be be *public*.

### Imput Format
You are not responsible for reading any input from stdin; a locked code stub will test your submission by calling the add method on an Adder object and passing it 2 integer parameters.

### Output Format
You are not responsible for printing anything to stdout.<br>
Your add method must return the sum of its parameters.

### Sample Output
> My superclass is: Arithmetic<br>
> 42 13 20
