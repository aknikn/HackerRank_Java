# 015: [Java Substring](https://www.hackerrank.com/challenges/java-substring/problem "https://www.hackerrank.com/challenges/java-substring/problem")
- level: Easy
- language: Java


## Description of the problem
Given a string, s, and two indices, start and end, print a substring consisting of all characters in the inclusive range from start to end -1.<br>
You'll find the String class' substring method helpful in completing this challenge.

### Imput Format
The first line contains a single string denoting.<br>
The second line contains two space-separated integers denoting the respective values of start and end.

### Constraints
- 0 ≦ |s| ≦ 100<br>
- 0 ≦ start < end ≦ n<br>
- String s consists of English alphabetic letters (i.e., [a-zA-Z]) only.

### Output Format
Print the substring in the inclusive range from start to end -1.

### Sample Input
>Helloworld<br>
> 3 7

### Sample Output
> lowo
